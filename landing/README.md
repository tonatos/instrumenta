# Bond Monitor Landing

Отдельный маркетинговый лендинг. **Не зависит** от `frontend/` и backend API.

## Стек

- Vite + TypeScript
- Свой CSS (без Tailwind)
- Playwright e2e в этой же папке

## Команды

```bash
cd landing
npm install
npm run dev          # http://127.0.0.1:5177
npm run build
npm run test:e2e
```

Или из корня: `task run:landing`
